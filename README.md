# 🎓 FMDD School - Online Training Platform

**FMDD School** is an online training platform developed as a final year project for the **FMDD Association**. Our mission is to support the digital transformation of education in Morocco by providing an inclusive, culturally relevant, and accessible learning environment.

---

## 🌟 Overview
In the context of Morocco’s evolving educational landscape, **FMDD School** addresses the gap in digital accessibility. Unlike international platforms, our solution focuses on:

* **Cultural Proximity:** Content and UX designed specifically for the local Moroccan context.
* **Accessibility:** Optimized for performance in **low-connectivity** and rural areas.
* **Civic Impact:** Making digital education a right for all Moroccans and international learners alike.

---

## 🚀 Key Features
* **For Instructors:** Independent course creation and management tools.
* **For Learners:** Seamless registration and a fluid, responsive learning interface.
* **Inclusivity:** Designed to accommodate diverse profiles with varying technical skills.

---

## 🛠️ Technical Journey
This project was a deep dive into **full-stack development**, moving from conception to deployment in a high-pressure environment.

* **Analysis & Design:** Strategic planning to ensure the platform meets real-world user needs.
* **Vibe Coding:** Embracing agile, intuitive development in real-world conditions.
* **Teamwork:** Developed by a 2-person team over a **2-month intensive period**.
* **Mentorship:** Developed under the guidance of **Mr. Hatim**, an IT Engineer with 15 years of industry experience.

---

## 💻 Tech Stack
* **Frontend:** React.js, Tailwind CSS, and Material UI
* **Backend:** PHP / Laravel
* **Database:** MySQL
* **Version Control:** Git & GitHub

---

## 📊 Development Process
The project was executed through four main phases:
1.  **Conception & Analysis:** Defining user personas and system architecture.
2.  **Tool Selection:** Choosing the right stack for scalability and performance.
3.  **Implementation:** Coding the core features and UI.
4.  **Deployment:** Transitioning the application to a live environment.
